## s 141

```markdown
**Basic info**
• Romaji: 
   - V-te kudasai
   - V-nai de kudasai
   - V-mashō
• Hiragana:
   - 〜てください
   - 〜ないでください
   - 〜ましょう
• Word type: Polite request / Prohibition / Suggestion forms

**Meaning**
• 〜てください = Please do … (polite request)
• 〜ないでください = Please don’t do … (polite negative request)
• 〜ましょう = Let’s … / Shall we … (suggestion, inclusive action)

**Formation**
1. Verb → て-form + ください
   Example: 食べます → 食べてください (*tabete kudasai* = please eat)
2. Verb → ない-form + でください
   Example: 行きます → 行かないでください (*ikanaide kudasai* = please don’t go)
3. Verb → ます-stem + ましょう
   Example: 行きます → 行きましょう (*ikimashō* = let’s go)

**Examples**
1. 水を飲んでください。  
   *Mizu o nonde kudasai.*  
   “Please drink water.”

2. タバコを吸わないでください。  
   *Tabako o suwanaide kudasai.*  
   “Please don’t smoke.”

3. 一緒に勉強しましょう。  
   *Issho ni benkyō shimashō.*  
   “Let’s study together.”

**Notes**
• 〜てください = polite but directive (used by teachers, doctors, instructions).  
• 〜ないでください = polite prohibition (used to stop someone politely).  
• 〜ましょう = inclusive suggestion, often used to encourage cooperation.  
```
