## 文法 (Grammar Points)

### 1. ～と思います (I think that...)
**Formation**: Plain form + と思います
- **Present affirmative**: いい → いいと思います
- **Present negative**: よくない → よくないと思います  
- **Past affirmative**: おいしかった → おいしかったと思います
- **Past negative**: おいしくなかった → おいしくなかったと思います

**Examples**:
- はなが いいと思います = I think flowers are good
- パーティーが いいと思います = I think a party would be good
- よろこぶと思います = I think (he) will be pleased

### 2. ～と言っていました (said that...)
**Formation**: Plain form + と言っていました
- **Used for reporting what someone said**
- Always uses plain form before と
- はなしたい → はなしたいと言っていました = (he) said he wants to talk
- すきじゃない → すきじゃないと言っていました = (he) said he doesn't like it
- しゅうしょくした → しゅうしょくしたと言っていました = (he) said he got a job

### 3. もう/まだ expressions
- **もう**: already
  - もうかった = already bought
  - もうしました = already did
- **まだ**: not yet, still
  - まだです = not yet
  - まだだ = not yet (plain form)

### 4. ～でも いいですか (is it okay if/even if...)
- もくようびでも いいですか = Is Thursday okay too?
- **Usage**: Asking if an alternative is acceptable

### 5. Gift-giving expressions
- ～から = from (someone)
- ～と～から = from ~ and ~
- これは～からです = This is from ~
- みんなから = from everyone

### 6. Polite gift-giving phrases
- あけても いいですか = Is it okay if I open it?
- たいせつにします = I'll treasure it
- すみません = I feel bad (but thank you)
