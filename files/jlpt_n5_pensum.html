Grammar — Core Patterns

文法

Particles

  • は · topic marker
  • が · subject marker
  • を · direct object
  • に · direction, time, indirect object
  • で · location of action, means
  • へ · direction
  • と · "and", "with"
  • の · possession, modification
  • も · "also"
  • から / まで · from / until
  • か · question marker
  • ね / よ · sentence-end particles

Copula & Sentence Frames

  • 〜です / 〜じゃありません · is / is not
  • 〜でした / 〜じゃありませんでした · past forms
  • 〜は〜です · basic A is B
  • 〜が あります / います · there is (thing / person)
  • 〜に〜が あります · existence at a place

Verb Forms (ます-form)

  • 〜ます / 〜ません · non-past affirmative / negative
  • 〜ました / 〜ませんでした · past forms
  • 〜ましょう / 〜ませんか · let's / shall we
  • 〜たい · want to do
  • 〜て form · connector, request, sequence
  • 〜てください · please do
  • 〜ています · progressive / state
  • 〜ないでください · please don't

Adjectives

  • い-adj + です · e.g. 高いです
  • い-adj → 〜くないです · negative
  • い-adj → 〜かったです · past
  • な-adj + です / じゃない · e.g. 静かです
  • い-adj → 〜くて / な-adj → 〜で · connecting form
  • 〜が好きです / 上手です · like / be good at

Question Words

  • 何 (なに/なん) · what
  • だれ / どなた · who
  • いつ · when
  • どこ · where
  • どう / どんな · how / what kind
  • いくら / いくつ · how much / how many
  • どれ / どちら · which
  • なぜ / どうして · why

Comparison & Connection

  • 〜より〜のほうが · more than
  • 〜の中で〜が一番 · most/best among
  • 〜と〜とどちらが · which of two
  • 〜から · because (reason)
  • 〜が · but (contrast)
  • そして / それから · and / then

Demonstratives

  • これ / それ / あれ / どれ · this / that / that over there / which
  • この / その / あの / どの + N · this/that/etc. + noun
  • ここ / そこ / あそこ / どこ · here / there / over there / where
  • こちら / そちら / あちら / どちら · polite directions

Counters & Time

  • 〜時 / 〜分 · o'clock / minutes
  • 〜日 / 〜月 / 〜年 · day / month / year
  • 〜つ (general) · 1〜10 native counter
  • 〜人 / 〜匹 / 〜本 / 〜枚 · people / animals / long / flat
  • 〜回 / 〜階 / 〜台 · times / floors / vehicles
